# custom serialiser and deserialiser

```python
import json

class MyObjectJsonEncoder(json.JSONEncoder):
  def default(self, obj):
    try:
      to_serialize = {
        'first': str(o.first),
        'size: str(o.size),
        ...
       }
       return to_serialize
     except AttributeError:
      return super().default(o)
```

```python
# client code

json_obj = json.dumps(object, cls=MyObjectJsonEnconder)
```
